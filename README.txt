========================================
  DAILY ROUTINE APP – Coventry, UK
  Install Guide (FREE – 5 minutes)
========================================

STEP 1 – Create a free GitHub account
  Go to: github.com → Sign Up (free)

STEP 2 – Create a new repository
  Click the + button → New repository
  Name it: my-routine
  Tick: "Add a README file"
  Click: Create repository

STEP 3 – Upload these 3 files
  Click "Add file" → Upload files
  Drag and drop:
    • index.html
    • sw.js
    • manifest.json
  Click: Commit changes

STEP 4 – Turn on GitHub Pages
  Click Settings → Pages (left sidebar)
  Under "Branch" select: main
  Click Save
  Wait 2 minutes
  Your app URL will be:
  https://YOUR-USERNAME.github.io/my-routine/

STEP 5 – Install on Android
  Open Chrome on your Android phone
  Go to your app URL above
  Tap the 3-dot menu (⋮)
  Tap "Add to Home screen"
  Tap Add
  The app icon appears on your home screen!

STEP 6 – Enable Notifications
  Open the app from your home screen
  Tap "Enable Notifications"
  Tap Allow
  You will now receive:
    ✅ 10-min warning before each prayer
    ✅ Alert at exact prayer time
    ✅ School drop-off reminder (8:05 AM)
    ✅ School pick-up reminder (3:00 PM)
    ✅ Dinner time, bath time, bedtime alerts

PRAYER TIMES
  Calculated automatically every day using
  Muslim World League method for Coventry
  52.41°N, 1.52°W — no internet needed.

SUPPORT
  If notifications don't appear:
  Android Settings → Apps → Chrome
  → Notifications → Allow
